## About the project

"회원제 중고거래 게시판 웹사이트" 라는 주제로 웹 프로젝트를 진행합니다.

프론트엔드를 구현한 코드를 저장한 레포지토리입니다.

[백엔드 레포지토리][1]

## Getting Started

### Prerequisites

- [npm in Node.js][2]

### How to execute server

1. Clone this repo.

   `git clone https://github.com/khu-net-web2022/web-frontend.git`

1. Install node packages.

   `cd web-frontend`

   `npm install`

1. Run the server.

   `npm start`

## Tech Stack / Library

- React.js, React Router
